About Appliance Alert AI

Appliance Alert AI is an intelligent real-time detection system built to prevent common household fire hazards. Using computer vision powered by YOLOv8, it identifies appliances like stoves, toasters, or irons left on without supervision and triggers alerts if no person is detected nearby.

Key Features

  • Real-time appliance and person detection via webcam or video
  • User login, registration, and secure access
  • Emergency alerts via WhatsApp API to trusted contacts
  • Upload images, directories, or use webcam input
  • Cloud-ready with support for deployment
  • Built-in security features and input validation
